Instructions

How do I add my citation?

  1. Fork the tethysplatform.github.io repository.
  2. Add your citation to the publications.html page following these guidelines:
    • Use the APA format.
    • List citations in approriate section (i.e.: papers about apps in the Tethys Apps section and papers related to platform devleopment in the "Tethys Platform" section).
    • List citations in Alphebetical order of author(s), chronological order if multiple for the same author(s).
  3. Submit a pull request with your changes.
  4. We will do our best to review in a timely manner. Be sure to watch the pull request thread for feedback.
